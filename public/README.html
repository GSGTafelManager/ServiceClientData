API Docs

Alle Tafeln listen

Request:

GET /online

Response:

{"online": [{"id": (id), "last": (Unix-Zeitpunkt der letzten Verbindung)}, ...]}

Befehl Senden

Request:

POST /send/(id)

Payload:

{"key": (Key), "exec": (Befehl), ...(Befehlspezifische Argumente)}

Befehle

open-url Website öffnen
Argumente:
"url": (url zum aufrufen), "mode": [popup/fullscreen/kiosk]

py Python-Code ausführen
Argumente:
"code": (code zum ausführen)

os cmd/Win-Code ausführen
Argumente:
"code": (code zum ausführen)

changeid ID ändern
Argumente:
"newid": (neue id)

close-app ALT+F4

Response:

{"status": [blocked/ignored/sent], "command": (Empfangener Befehl)}
